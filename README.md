# Titanic Dataset Exploratory Data Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic Passenger Dataset using Google Sheets. The objective is to explore the dataset, identify patterns, visualize key trends, and generate insights related to passenger survival.

## Dataset

* Dataset Name: Titanic Passenger Dataset
* Total Records: 891 Passengers
* Features Used:

  * Survived
  * Pclass
  * Sex
  * Age
  * Fare

## Analysis Performed

* Data Exploration
* Survival Distribution Analysis
* Passenger Class Analysis
* Gender Distribution Analysis
* Age Distribution Analysis
* Insight Generation

## Visualizations

1. Survival Distribution Pie Chart
2. Passenger Class Bar Chart
3. Gender Distribution Chart
4. Age Distribution Histogram

## Key Insights

* More passengers did not survive than survived.
* Male passengers were more than female passengers.
* Female passengers had a higher survival rate.
* Third-class passengers formed the largest group.
* Passenger class influenced survival chances.
* Most passengers were between 20 and 40 years old.
* Ticket fares varied significantly.
* Some Age values were missing in the dataset.

## Conclusion

The analysis revealed that gender, passenger class, and age played important roles in passenger survival. Female passengers and first-class passengers had higher survival rates, while third-class passengers formed the majority of travelers.
